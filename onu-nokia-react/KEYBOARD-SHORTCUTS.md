# Atalhos de Teclado

## Navegação de Componentes

| Atalho     | Ação                       |
| ---------- | -------------------------- |
| `Ctrl + 1` | Abrir Pesquisar Cliente    |
| `Ctrl + 2` | Abrir Provisionar Cliente  |
| `Ctrl + 3` | Abrir Bridge ONT Nokia     |
| `Ctrl + 4` | Abrir Configurar Wi-Fi     |
| `Ctrl + 5` | Abrir Configurar Telefone  |
| `Ctrl + 6` | Abrir Alterar VLAN PPPOE   |
| `Ctrl + 7` | Abrir Alterar Senha Web    |
| `Ctrl + 8` | Abrir Conferência de Caixa |
| `Ctrl + 9` | Abrir Outras Opções        |

## Interface

| Atalho     | Ação                 |
| ---------- | -------------------- |
| `Ctrl + B` | Abrir/Fechar Sidebar |
| `Esc`      | Fechar Sidebar       |

---

## Dicas

- Os atalhos funcionam em qualquer parte da aplicação
- Use `Esc` para fechar a sidebar rapidamente
- Os números dos atalhos correspondem à ordem dos itens no menu
